# Kering S.A. — Financial Analysis

A financial analysis of Kering Group focused on one key question:
**To what extent does Gucci drive Kering’s overall profitability?**

Between 2021 and 2025, Kering’s revenue declined by 16.8%, operating income fell by 79%, and net income dropped from €3.6B to €72M. Gucci, contributing 41% of revenue but 59% of operating profit, accounted for the majority of this decline.

---

## Overview

This project analyses Kering’s financial performance across revenue, margins, returns, and brand-level contribution, with a focus on identifying the key drivers of profitability and performance deterioration.

---

## Key Components

| Component              | Description                                                                             |
| ---------------------- | --------------------------------------------------------------------------------------- |
| Financial Statements   | Consolidated income statement, balance sheet, and cash flow (2020–2025)                 |
| Common Size Analysis   | Margin and asset structure trends expressed as percentages over time                    |
| Brand-Level Analysis   | Revenue and operating profit contribution by brand, including margin comparison         |
| Performance Trends     | Revenue, cost structure, and margin evolution with focus on Gucci’s impact              |
| Returns and Efficiency | ROE, DuPont analysis, ROIC, and working capital metrics, including cash conversion cycle|
| Forecasting            | Bottom-up brand-level revenue projections with defined assumptions                      |
| Scenario Analysis      | Impact of different Gucci recovery outcomes on group profitability                      |
| Key Findings           | Summary of core insights supported by charts and KPIs                                   |
| Strategic Implications | Conclusions and recommendations based on financial analysis                             |

---

## Key Insight

Kering’s financial performance is highly concentrated. Changes in Gucci’s performance have a disproportionate impact on group profitability, making recovery dependent on both brand repositioning and portfolio balance.
